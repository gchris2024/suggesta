## Landing / Landing Mobile

Landing page is what the user sees when they open the site.

## Login / Login Mobile

Login page is what appears when the user clicks "Log In"

## Register / Register Mobile

Register page is what appears when the user clicks "Register"

## Home

The Home page is the first page the user sees upon successfully logging in / registering

## Recommendations

The Recommendations page appears when the users clicks the "Generate Recommendations" button

## Saved

The Saved page appears when the users clicks the "My Lists" button
